![](images/v2rayn订阅导入003.png)

![](images/v2rayn订阅导入004.png)

![](images/v2rayn订阅导入005.png)

![](images/v2rayn订阅导入006.png)

![](images/v2rayn订阅导入008.png)



可以看到图标变红了，可以翻墙了。红色是正在使用代理，蓝色是没有使用代理
